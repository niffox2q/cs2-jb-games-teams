### Модуль Teams Game для Jailbreak Core  
Добавляет мини-игру с разделением живых заключённых на 2–4 случайные команды с визуальной окраской игроков.
Игру можно запустить через меню Начальника тюрьмы.
Игроки распределяются автоматически после выбора количества команд в меню Начальника Тюрьмы. При завершении раунда или смерти Начальника Тюрьмы все эффекты сбрасываются.

### Установка:
Распаковать архив в `game/csgo/addons/`

### Требования:
- [Jailbreak Core](https://discord.gg/WkTwuKe8zy)
- [Utils](https://github.com/Pisex/cs2-menus)
